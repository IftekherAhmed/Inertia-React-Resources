## Setting Up a New Resource

To create a new resource (e.g., "Post"), follow these steps:

### 1. Create Migration
```bash
php artisan make:migration create_posts_table
```

Example migration structure:
```php
Schema::create('posts', function (Blueprint $table) {
    $table->id();
    $table->string('title');
    $table->text('content');
    $table->string('image_path')->nullable(); // For file uploads
    $table->boolean('published')->default(false);
    $table->timestamps();
});
```

### 2. Create Model
```bash
php artisan make:model Models/Post
```

Add fillable fields and relationships:
```php
class Post extends Model
{
    protected $fillable = [
        'title',
        'content',
        'image_path',
        'published',
    ];

    protected $casts = [
        'published' => 'boolean',
        'created_at' => 'datetime',
        'updated_at' => 'datetime',
    ];
    
    // Accessor for full image URL
    public function getImageUrlAttribute()
    {
        return $this->image_path ? asset('storage/' . $this->image_path) : null;
    }
}
```

### 3. Create Factory (for seeding/testing)
```bash
php artisan make:factory PostFactory
```

Example factory:
```php
public function definition(): array
{
    return [
        'title' => fake()->sentence(),
        'content' => fake()->paragraphs(3, true),
        'published' => fake()->boolean(),
    ];
}
```

### 4. Create Seeder
```bash
php artisan make:seeder PostSeeder
```

Example seeder:
```php
public function run(): void
{
    Post::factory()->count(50)->create();
}
```

### 5. Run Migration and Seed
```bash
php artisan migrate --seed
```

## Database Layer

### Model Scopes for Data Operations
Implement query scopes in your model for reusable query logic:

```php
// Search scope - use when you need to search across multiple fields
public function scopeSearch(Builder $query, ?string $search): Builder
{
    if (empty($search)) {
        return $query;
    }

    return $query->where(function (Builder $q) use ($search): void {
        $q->where('title', 'like', "%{$search}%")
          ->orWhere('content', 'like', "%{$search}%");
    });
}

// Filter by published status - use when you need boolean filtering
public function scopePublished(Builder $query, ?bool $published): Builder
{
    if (is_null($published)) {
        return $query;
    }

    return $query->where('published', $published);
}

// Filter by date range - use when you need date filtering
public function scopeDateRange(Builder $query, ?string $startDate, ?string $endDate): Builder
{
    if ($startDate) {
        $query->whereDate('created_at', '>=', $startDate);
    }
    
    if ($endDate) {
        $query->whereDate('created_at', '<=', $endDate);
    }
    
    return $query;
}

// Sort scope - use when you need sorting functionality
public function scopeSortBy(Builder $query, string $sortBy = 'id', string $sortDirection = 'desc'): Builder
{
    $allowedSortFields = ['id', 'title', 'created_at', 'updated_at'];
    $sortBy = in_array($sortBy, $allowedSortFields) ? $sortBy : 'id';
    $sortDirection = in_array(strtolower($sortDirection), ['asc', 'desc']) ? $sortDirection : 'desc';

    return $query->orderBy($sortBy, $sortDirection);
}

// Combined filter and sort scope - use this in your index method
public function scopeFilteredAndSorted(Builder $query, array $filters): Builder
{
    return $query->search($filters['search'] ?? null)
                 ->published($filters['published'] ?? null)
                 ->dateRange($filters['start_date'] ?? null, $filters['end_date'] ?? null)
                 ->sortBy($filters['sort_by'] ?? 'id', $filters['sort_direction'] ?? 'desc');
}
```

#### Index Method (Listing with All Features)

```php
// Use this when you need to list resources with search, filter, sort, and pagination
public function index(Request $request)
{
    // Extract filters from request
    $filters = [
        'search' => $request->input('search'),
        'published' => $request->input('published'),
        'start_date' => $request->input('start_date'),
        'end_date' => $request->input('end_date'),
        'per_page' => $request->input('per_page', 10),
        'sort_by' => $request->input('sort_by', 'id'),
        'sort_direction' => $request->input('sort_direction', 'desc'),
    ];

    $page = $request->input('page', 1);
    $perPage = $filters['per_page'];

    // Apply filters and sorting using model scopes
    $query = Post::filteredAndSorted($filters);
    
    // Handle pagination
    if ($perPage === 'all' || $perPage == 0) {
        $results = $query->get();
        $pagination = [
            'current_page' => 1,
            'last_page' => 1,
            'per_page' => 'all',
            'total' => $results->count(),
        ];
        $postList = $results;
    } else {
        $results = $query->paginate($perPage, ['*'], 'page', $page);
        $pagination = [
            'current_page' => $results->currentPage(),
            'last_page' => $results->lastPage(),
            'per_page' => $results->perPage(),
            'total' => $results->total(),
        ];
        $postList = $results->items();
    }

    return Inertia::render('Post/Index', [
        'post_list' => $postList,
        'pagination' => $pagination,
        'filters' => $filters,
    ]);
}
```

#### Store Method (Create with File Upload)
```php
// Use this when you need to create a resource with file upload
public function store(PostRequest $request)
{
    try {
        $data = new Post();
        
        // Handle file upload if present
        if ($request->hasFile('image')) {
            $imagePath = $request->file('image')->store('posts', 'public');
            $data->image_path = $imagePath;
        }
        
        $this->fillRequestData($request, $data);
        $data->save();

        return redirect()->route('post.index')->with('success', 'Post created successfully!');
    } catch (Exception $exception) {
        return back()->with('error', 'An error occurred: ' . $exception->getMessage())->withInput();
    }
}
```

#### Update Method (Update with File Upload)
```php
// Use this when you need to update a resource with file upload
public function update(PostRequest $request, Post $post)
{
    try {
        // Handle file upload if present
        if ($request->hasFile('image')) {
            // Delete old image if exists
            if ($post->image_path && Storage::disk('public')->exists($post->image_path)) {
                Storage::disk('public')->delete($post->image_path);
            }
            
            $imagePath = $request->file('image')->store('posts', 'public');
            $post->image_path = $imagePath;
        }
        
        $this->fillRequestData($request, $post);
        $post->save();

        return redirect()->route('post.index')->with('success', 'Post updated successfully!');
    } catch (Exception $exception) {
        return back()->with('error', 'An error occurred: ' . $exception->getMessage())->withInput();
    }
}
```

## Frontend Implementation

### 1. Create Page Components
Create directory `resources/js/pages/Post/` with these files:

#### Index.tsx (Listing Page)
```tsx
import { Card, CardContent, CardFooter } from '@/components/ui/card';
import { Select, SelectContent, SelectItem, SelectTrigger, SelectValue } from '@/components/ui/select';
import AppLayout from '@/layouts/app-layout';
import { Head, router, usePage } from '@inertiajs/react';
import { Search, CalendarRange, ListFilter, RotateCcw, Pencil, Trash2, Eye } from 'lucide-react';
import { useState, useCallback } from 'react';
import Pagination from '@/components/custom_components/Pagination';
import { Input } from '@/components/ui/input';
import LinkButton from '@/components/LinkButton';
import { Button } from '@/components/ui/button';

export default function Index() {
    const { props } = usePage();
    const { post_list, pagination, filters } = props as any;

    // Local state for filters and UI interactions
    const [searchQuery, setSearchQuery] = useState(filters.search || '');
    const [publishedFilter, setPublishedFilter] = useState(filters.published || '');
    const [startDate, setStartDate] = useState(filters.start_date || '');
    const [endDate, setEndDate] = useState(filters.end_date || '');
    const [perPage, setPerPage] = useState(filters.per_page || 10);
    const [currentPage, setCurrentPage] = useState(pagination.current_page || 1);

    // Fetch posts with given filters via Inertia
    const fetchPosts = useCallback((params: Partial<any>) => {
        const query = {
            search: params.search ?? searchQuery,
            published: params.published ?? publishedFilter,
            start_date: params.start_date ?? startDate,
            end_date: params.end_date ?? endDate,
            per_page: params.per_page ?? perPage,
            page: params.page ?? currentPage,
        };

        router.get('/post', query, {
            preserveState: true,
            replace: true,
        });
    }, [searchQuery, publishedFilter, startDate, endDate, perPage, currentPage]);

    // Handle search input
    const handleSearchChange = (e: React.ChangeEvent<HTMLInputElement>) => {
        setSearchQuery(e.target.value);
        fetchPosts({ search: e.target.value, page: 1 });
    };

    // Handle published filter
    const handlePublishedChange = (value: string) => {
        setPublishedFilter(value);
        fetchPosts({ published: value, page: 1 });
    };

    // Handle date filters
    const handleStartDateChange = (e: React.ChangeEvent<HTMLInputElement>) => {
        setStartDate(e.target.value);
        fetchPosts({ start_date: e.target.value, page: 1 });
    };

    const handleEndDateChange = (e: React.ChangeEvent<HTMLInputElement>) => {
        setEndDate(e.target.value);
        fetchPosts({ end_date: e.target.value, page: 1 });
    };

    // Handle per-page change
    const handlePerPageChange = (value: string) => {
        const perPage = value === 'all' ? 0 : Number(value);
        setPerPage(perPage);
        fetchPosts({ per_page: perPage, page: 1 });
    };

    // Handle pagination
    const handlePageChange = (page: number) => {
        setCurrentPage(page);
        fetchPosts({ page });
    };

    // Reset filters
    const resetTable = () => {
        setSearchQuery('');
        setPublishedFilter('');
        setStartDate('');
        setEndDate('');
        setPerPage(10);
        setCurrentPage(1);
        fetchPosts({ search: '', published: '', start_date: '', end_date: '', per_page: 10, page: 1 });
    };

    // Delete handler
    const handleDelete = (e: any) => {
        if (!confirm('Are you sure you want to delete this post?')) {
            e.preventDefault();
        }
    };

    return (
        <AppLayout>
            <Head title="Posts" />

            <div className="flex h-full flex-1 flex-col gap-4 rounded-xl p-4">
                <div className="flex items-center justify-between">
                    <h1 className="text-2xl font-bold">Posts</h1>
                    <LinkButton href="/post/create">+ New Post</LinkButton>
                </div>

                <Card>
                    <CardContent>
                        {/* Filter section */}
                        <div className="mb-4 flex flex-wrap gap-4">
                            {/* Search input */}
                            <div className='relative'>
                                <div className="absolute inset-y-0 left-0 pl-3 flex items-center pointer-events-none">
                                    <Search className="h-4 w-4 text-gray-400" />
                                </div>
                                <Input
                                    type="text"
                                    placeholder="Search posts..."
                                    value={searchQuery}
                                    onChange={handleSearchChange}
                                    className="rounded border px-2 py-1 pl-9"
                                />
                            </div>

                            {/* Published filter */}
                            <div>
                                <Select
                                    value={publishedFilter}
                                    onValueChange={handlePublishedChange}
                                >
                                    <SelectTrigger>
                                        <ListFilter className="h-4 w-4 text-gray-400" />
                                        <div className="pl-2">
                                            <SelectValue placeholder="All Status" />
                                        </div>
                                    </SelectTrigger>
                                    <SelectContent>
                                        <SelectItem value="">All Status</SelectItem>
                                        <SelectItem value="1">Published</SelectItem>
                                        <SelectItem value="0">Draft</SelectItem>
                                    </SelectContent>
                                </Select>
                            </div>

                            {/* Date range filters */}
                            <div className='relative'>
                                <div className="absolute inset-y-0 left-0 pl-3 flex items-center pointer-events-none">
                                    <CalendarRange className="h-4 w-4 text-gray-400" />
                                </div>
                                <Input
                                    type="date"
                                    value={startDate}
                                    onChange={handleStartDateChange}
                                    className="rounded border px-2 py-1 pl-9"
                                    placeholder="Start date"
                                />
                            </div>

                            <div className='relative'>
                                <div className="absolute inset-y-0 left-0 pl-3 flex items-center pointer-events-none">
                                    <CalendarRange className="h-4 w-4 text-gray-400" />
                                </div>
                                <Input
                                    type="date"
                                    value={endDate}
                                    onChange={handleEndDateChange}
                                    className="rounded border px-2 py-1 pl-9"
                                    placeholder="End date"
                                />
                            </div>

                            {/* Per page dropdown */}
                            <div>
                                <Select
                                    value={perPage === 0 ? 'all' : perPage.toString()}
                                    onValueChange={handlePerPageChange}
                                >
                                    <SelectTrigger>
                                        <ListFilter className="h-4 w-4 text-gray-400" />
                                        <div className="pl-2">
                                            <SelectValue placeholder={perPage === 0 ? 'All' : `${perPage} per page`} />
                                        </div>
                                    </SelectTrigger>
                                    <SelectContent>
                                        {['5', '10', '25', '50', '100'].map(size => (
                                            <SelectItem key={size} value={size}>{size} per page</SelectItem>
                                        ))}
                                        <SelectItem value="all">All</SelectItem>
                                    </SelectContent>
                                </Select>
                            </div>

                            {/* Reset filters button */}
                            <div className="ml-auto">
                                <Button onClick={resetTable}
                                    className="bg-white text-gray-800 hover:bg-gray-100 dark:bg-gray-800 dark:text-white dark:hover:bg-gray-700 border border-gray-200 dark:border-gray-600">
                                    <RotateCcw size={16} /> Reset
                                </Button>
                            </div>
                        </div>

                        {/* Post list */}
                        <div className="gap-4 grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3">
                            {post_list.map((post: any) => (
                                <div key={post.id} className="border rounded-lg p-4">
                                    {post.image_path && (
                                        <img 
                                            src={`/storage/${post.image_path}`} 
                                            alt={post.title} 
                                            className="w-full h-32 object-cover rounded mb-2"
                                        />
                                    )}
                                    <h2 className="text-lg font-semibold">{post.title}</h2>
                                    <p className="text-sm text-gray-500">
                                        {new Date(post.created_at).toLocaleDateString()}
                                    </p>
                                    <div className="flex justify-end space-x-2 mt-2">
                                        <Link href={`/post/${post.id}`} className="text-gray-500">
                                            <Eye size={16} />
                                        </Link>
                                        <Link href={`/post/${post.id}/edit`} className="text-blue-500">
                                            <Pencil size={16} />
                                        </Link>
                                        <Link
                                            as="button"
                                            method="delete"
                                            href={`/post/${post.id}`}
                                            className="text-red-500"
                                            onClick={handleDelete}>
                                            <Trash2 size={16} />
                                        </Link>
                                    </div>
                                </div>
                            ))}
                        </div>
                    </CardContent>

                    <CardFooter>
                        <div className="space-y-2">
                            <p className="text-muted-foreground">You have {pagination.total} posts.</p>
                            {perPage !== 0 && (
                                <Pagination
                                    currentPage={pagination.current_page}
                                    totalPages={pagination.last_page}
                                    onPageChange={handlePageChange}
                                />
                            )}
                        </div>
                    </CardFooter>
                </Card>
            </div>
        </AppLayout>
    );
}
```

#### Create.tsx (Create Form with File Upload)

```tsx
import InputError from '@/components/input-error';

import { Input } from '@/components/ui/input';

import { Label } from '@radix-ui/react-label';

export default function Create() {
    const { data, setData, post, processing, errors } = useForm({
        title: '',
        content: '',
        image: null as File | null,

    });

            <Head title="Create Post" />

                    <h1 className="text-2xl font-bold">Create Post</h1>

                                <Label htmlFor='title'>Title</Label>

    );
```

## Sorting Operations

### Backend Implementation

```php
public function scopeSortBy(Builder $query, string $sortBy = 'id', string $sortDirection = 'desc'): Builder
{
    $allowedSortFields = ['id', 'title', 'created_at', 'updated_at'];
    $sortBy = in_array($sortBy, $allowedSortFields) ? $sortBy : 'id';
    $sortDirection = in_array(strtolower($sortDirection), ['asc', 'desc']) ? $sortDirection : 'desc';

}
```
